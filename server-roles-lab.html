📋 مرجع سريع لأدوار الخادم

🌐 خادم DNS
يترجم أسماء النطاقات إلى عناوين IP. المنفذ 53 UDP/TCP.
📡 خادم DHCP
يعيّن عناوين IP تلقائيًا للعملاء. المنافذ 67/68 UDP.
👥 Active Directory
مصادقة وتفويض مركزيان. LDAP المنفذ 389، Kerberos 88.
🌍 خادم الويب
يخدم صفحات HTTP/HTTPS. المنفذ 80 و443.
📧 خادم البريد
الإرسال/الاستقبال. SMTP 25، IMAP 143، POP3 110.
🗄️ خادم قواعد البيانات
SQL/NoSQL. MySQL 3306، MSSQL 1433.
📁 خادم الملفات
تخزين مشترك. SMB 445، NFS 2049.
⏰ خادم NTP
مزامنة ساعات النظام. المنفذ 123 UDP.
