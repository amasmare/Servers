📋 Server Roles Quick Reference

🌐 DNS Server
Translates domain names to IP addresses. Port 53 UDP/TCP.
📡 DHCP Server
Automatically assigns IP addresses to network clients. Ports 67/68 UDP.
👥 Active Directory
Centralized authentication and authorization. LDAP port 389, Kerberos 88.
🌍 Web Server
Serves HTTP/HTTPS web pages. Port 80 HTTP, 443 HTTPS.
📧 Mail Server
Sends/receives email. SMTP port 25, IMAP 143, POP3 110.
🗄️ Database Server
SQL/NoSQL data storage. MySQL port 3306, MSSQL 1433.
📁 File Server
Shared file storage. SMB port 445, NFS 2049.
⏰ NTP Server
Synchronizes system clocks. Port 123 UDP.
